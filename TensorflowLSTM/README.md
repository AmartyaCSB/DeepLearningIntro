# TensorflowLSTM

This is the code for generating generate new text, based on the text learned from wiki. We will use an LSTM RNN in TensorFlow.


Language
============
* Python 3


Version
============
tensorflow 1.5.0
python 3.5.2


Usage
============

Run on CPU with command:

    python generator.py [# of epochs]
